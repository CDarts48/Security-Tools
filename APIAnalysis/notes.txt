The HTTP specifications defines a special method that only exists to information about a particular API's verbs. 
This method is called OPTIONS, which should be our to go-to when preforming recon against an API
We can make a curl request in the terminal.
curl -i -X OPTIONS https://api.mega-bank.com/users/123
